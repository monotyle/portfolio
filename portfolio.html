01 · REST API

fastapi-notes-api

REST API для управления заметками. Реализованы четыре эндпоинта — получение всех заметок, создание, обновление и удаление по ID. Данные хранятся in-memory. Встроенная документация через Swagger UI доступна по /docs.

fastapi pydantic uvicorn python 3.10+ swagger ui
Открыть на GitHub
МетодПутьОписание
GET/notesПолучить все заметки
POST/notesСоздать заметку
PUT/notes/{id}Обновить по ID
DELETE/notes/{id}Удалить по ID

02 · REST API

fastapi-url-shortener

Сервис сокращения ссылок. Генерирует уникальные 6-значные коды, переиспользует существующие коды для одинаковых URL, выполняет редирект (307) и позволяет удалять ссылки. Данные хранятся in-memory.

fastapi pydantic uvicorn python 3.10+ secrets
Открыть на GitHub
МетодПутьОписание
POST/shortenСоздать короткую ссылку
GET/{code}Редирект (307)
GET/info/{code}Информация о коде
DELETE/delete/{code}Удалить ссылку
